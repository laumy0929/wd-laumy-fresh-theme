# Laumy Fresh Theme

一个简洁、响应式的WordPress主题，专为技术博客和内容网站设计。

## 🌟 主题特色

### 设计理念
- **简洁美观**：采用现代简约设计风格，专注于内容阅读体验
- **响应式布局**：完美适配桌面端、平板和移动设备
- **深色模式**：支持一键切换深色/浅色主题
- **高效性能**：优化的CSS和JavaScript，确保快速加载

### 核心功能
- 📱 **三栏布局**：左侧分类导航 + 中间内容区域 + 右侧个人信息
- 🔍 **智能搜索**：顶部居中搜索框，快速查找内容
- 📂 **分类管理**：支持多级分类展开/收起，显示文章数量
- 📖 **文章目录**：自动生成多级目录，支持滚动高亮
- 💬 **评论系统**：完整的WordPress评论功能
- 📊 **数据统计**：显示文章数量和阅读量统计
- 🎨 **主题切换**：一键切换深色/浅色模式

## 🚀 安装方法

### 方法一：WordPress后台安装
1. 下载主题压缩包
2. 在WordPress后台进入 `外观 > 主题`
3. 点击 `添加新主题` > `上传主题`
4. 选择下载的压缩包文件
5. 点击 `现在安装` 并激活主题

### 方法二：FTP上传
1. 解压主题文件到本地
2. 通过FTP上传到 `/wp-content/themes/` 目录
3. 在WordPress后台激活主题

## ⚙️ 主题配置

### 1. 自定义设置
进入 `外观 > 自定义` 进行以下配置：

#### 作者信息设置
- **头像**：上传个人头像图片
- **昵称**：设置显示名称
- **职业**：添加职业标签
- **个性签名**：个人简介描述

#### 布局调整
- **侧边栏宽度**：可调整左右侧边栏宽度
- **内容区域**：设置最大宽度和间距
- **颜色主题**：自定义主色调和边框色

### 2. 菜单配置
- 在 `外观 > 菜单` 中创建主菜单
- 分类菜单会自动显示在左侧边栏
- 支持多级分类结构

### 3. 小工具配置
- 右侧边栏支持添加WordPress小工具
- 可自定义显示内容和顺序

## 📱 响应式设计

### 桌面端 (>1200px)
- 三栏完整布局
- 左侧分类导航：260px
- 右侧个人信息：300px
- 中间内容区域：自适应宽度

### 平板端 (992px-1200px)
- 侧边栏宽度自动调整
- 保持三栏布局结构

### 移动端 (<768px)
- 单列布局，内容垂直排列
- 隐藏网站标题和部分导航链接
- 搜索框居中显示
- 保留主题切换功能

## 🎨 自定义样式

### CSS变量
主题使用CSS变量，便于自定义：

```css
:root {
  --header-height: 56px;
  --sidebar-left-width: 260px;
  --sidebar-right-width: 300px;
  --container-max-width: 1800px;
  --gap: 14px;
  --color-bg: #ffffff;
  --color-text: #222;
  --color-muted: #666;
  --color-primary: #3498db;
  --color-border: #e6e6e6;
}
```

### 深色模式
深色模式自动调整所有颜色：
- 背景色：`#181a1b`
- 文字色：`#eceff1`
- 卡片背景：`#202225`

## 🎯 使用技巧

### 1. 优化文章缩略图
- 设置特色图片（推荐尺寸：160×100px）
- 或在文章开头插入图片
- 主题会自动提取第一张图片作为缩略图

### 2. 分类管理建议
- 使用清晰的分类名称
- 合理设置分类层级（建议不超过3级）
- 定期整理和合并相似分类

### 3. 性能优化
- 启用WordPress缓存插件
- 压缩图片文件大小
- 定期清理数据库

## 🐛 常见问题

### Q: 分类不显示或显示异常？
A: 检查是否在 `外观 > 菜单` 中设置了主菜单，分类会自动从主菜单中提取。

### Q: 深色模式切换不生效？
A: 确保浏览器支持localStorage，清除浏览器缓存后重试。

### Q: 文章目录不显示？
A: 检查文章内容是否包含H2-H5标题，目录基于这些标题生成。

### Q: 移动端显示异常？
A: 主题已针对移动端优化，如仍有问题请检查是否有其他插件冲突。
